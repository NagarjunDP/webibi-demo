# Webibi Demo Generator
> Your personalized website, generated in 60 seconds from a brand logo.

Webibi Demo is a next-generation web application designed for digital agencies to create instant, personalized, and high-converting website demonstrations for local businesses during live pitches.

---

## 🚀 Key Features

* **Instant Color DNA Extraction**: Upload a brand logo, and the app will automatically extract the core color palette.
* **Google Places Enrichment**: Automatically retrieves live Google Maps reviews, ratings, and customer counts to inject social proof directly into the template.
* **Dynamic Sector Templates**: Built-in bespoke HTML templates for Restaurants, Gyms, Salons, Clinics, and Law Firms.
* **Simulated Analytics & Sharing**: Provides a live preview with simulated real-time client analytics to close deals faster.

## 🛠️ Getting Started

### 1. Install Dependencies
```bash
npm install
```

### 2. Set Up Environment Variables
Create a `.env` or `.env.local` file in the root:
```env
GOOGLE_PLACES_API_KEY=your_google_places_api_key
```

### 3. Run Development Server
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the application.
